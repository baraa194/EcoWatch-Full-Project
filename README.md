# 🌱 EcoWatch System

EcoWatch is a smart environmental platform designed to encourage recycling behavior, reward sustainable actions, and connect users with volunteering opportunities. The system uses AI-powered classification to help users identify recyclable materials and contribute to reducing waste in their community.

---

## 🚀 Project Overview

EcoWatch aims to:
- Reduce environmental waste
- Encourage recycling habits
- Reward users with points for eco-friendly actions
- Connect individuals with volunteering opportunities
- Use AI to analyze and classify recyclable materials

---

## ♻️ Recycling Module

The recycling system is based on two main options:

### 1. Street Image Classification (AI-Based)
Users can upload an image taken from the street or environment.
- The system uses AI to detect and classify waste materials
- Examples: plastic, paper, glass, metal, organic waste
- Users receive points based on the type and quantity of detected materials

### 2. Home Materials Submission
Users can manually submit recyclable materials available at home.
- Select or input materials (plastic bottles, cardboard, cans, etc.)
- The system calculates estimated weight/quantity
- Points are awarded based on material type and amount

## 🧠 AI Integration

- Image classification model for waste detection
- Automatic material recognition from uploaded photos

## 🚨 Environmental Reporting Module

EcoWatch also includes a reporting system that allows users to report environmental issues directly to relevant authorities.

### 📝 How it works:
- Users can submit a report about any environmental problem they notice
  - Examples: garbage accumulation, illegal dumping, pollution areas, blocked drainage, etc.
- Each report can include:
  - Description of the issue
  - Location (if available)
  - Image evidence (optional but recommended)

### 🏛️ Authority Handling:
- Reports are sent to the responsible authorities for review and action
- Authorities can update the status of each report

### 📊 Report Status Tracking:
Users can track the status of their submitted reports:
- 🟡 **Pending** → Report has been received and is under review  
- 🔵 **In Progress** → Authorities are currently handling the issue  
- 🟢 **Resolved** → The issue has been successfully fixed  

### 🔔 Notifications:
- Users receive updates when the status of their report changes
- Ensures transparency and engagement between users and authorities
